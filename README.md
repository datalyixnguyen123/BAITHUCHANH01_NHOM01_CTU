## Nội dung thực hành

### Phần 1: Cấu hình phần cứng & phần mềm với AI
**Nhiệm vụ 1.1 – Tìm hiểu cấu hình phần cứng**
- Kiểm tra CPU, RAM, ổ cứng bằng Task Manager/System Information.
- Tham khảo hướng dẫn từ AI.
- Kết quả: File văn bản + ảnh chụp màn hình.

**Nhiệm vụ 1.2 – Cấu hình phần mềm**
- Cài đặt và cấu hình Notepad++ để viết code Python.
- Tạo file Python in ra dòng: `Hello, AI!`
- Kết quả: File `.py` + ảnh chụp màn hình.

---

### Phần 2: Giải quyết sự cố kỹ thuật với AI
**Nhiệm vụ 2.1 – Sự cố Wi-Fi**
- Sử dụng AI để tìm cách khắc phục lỗi Wi-Fi trên Windows 10.
- Thực hiện các lệnh mạng cơ bản (`ipconfig /release`, `ipconfig /renew`).
- Kết quả: Báo cáo ngắn + ảnh chụp màn hình.

**Nhiệm vụ 2.2 – Lỗi phần mềm**
- Xử lý lỗi Notepad++ không mở được file Python lớn.
- Áp dụng giải pháp AI đề xuất.
- Kết quả: Báo cáo mô tả lỗi và giải pháp.

---

### Phần 3: Ứng dụng AI trong học tập
**Nhiệm vụ 3.1 – Tạo infographic với Canva AI**
- Chủ đề: *Ứng dụng AI trong Công nghệ thông tin*.
- Xuất file PNG và gán giấy phép **CC BY**.
- Ghi rõ: *Infographic được tạo bằng Canva AI*.

**Nhiệm vụ 3.2 – Đánh giá đạo đức AI**
- Viết đoạn văn 100–150 từ về việc sử dụng AI minh bạch và có trách nhiệm.
- Tham khảo DigComp 2.2.
- Kết quả: File văn bản phân tích đạo đức.
